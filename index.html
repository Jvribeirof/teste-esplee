Duvidas frequentes

Seres humanos nunca são iguais aos outros. Mesmo com características semelhantes aos outros, sempre possuem detalhes que os diferenciam. Isso também serve para as marcas, elas nasceram para criar desigualdade entre as empresas.

Vivemos em um mundo onde a autenticidade virou incomum. Vários seguem o fluxo das coisas e sem perceber, fazem do mesmo. Ter uma marca é para quem quer sair desse fluxo que não caba.

Depende. Cada empresa tem seus próprios problemas para ser resolvidos. Após uma reunião para descobrir os tais problemas, enviaremos uma proposta do projeto.

A Esplee é um estúdio de design que projeta identidades visuais aliadas a um auxílio do branding especialmente para pequenas empresas que buscam se diferenciar do mercado explorando o que ela tem de único.
